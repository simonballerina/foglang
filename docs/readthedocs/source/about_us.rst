About us
========

What is Foglang?
----------------

Foglang is a modern, sleek and revolutionizing programming language perfect for *your* next project.
It is developed and maintained mainly by Simonballerina representing Stem-Klubben.
Foglang is currently being interpreted using C. The language is classified as strong and dynamically typed.

Origin story
------------

Foglang was created by `Simonballerina <https://www.github.com/simonballerina/>`_ in early February 2026.
The early versions were quite basic and included some unorthodox syntax.
Versions zero and one were quickly abandoned and, within two weeks of launch,
Foglang2 was established as the prominent version of the language.
`Hannes <https://www.github.com/handej08/>`_ created this readthedocs page in early March.
He later went on to help Simonballerina with the general Foglang development.
